# remember-helper
用这个软件可以帮助记忆，比如看见英语单词帮助回忆翻译等
## 用法：
1. 新建excel表
2. 第一列写不会的东西，例如abandon
3. 第二列写数字，1代表会，2代表不会
4. 第三列写答案，因为懒所以不想改到第二列写答案

|   第一行不写东西         |             |          |
| :----: | :----: | :----: |
| abandon        | 2           |      我不会   |
| ability        | 1           |      还是不会   |
| able           | 2           |   就是不会  |
|abnormal        | 2           |   咋都不会   |
