# remember-helper
用这个软件可以帮助记忆，比如看见英语单词帮助回忆翻译等
## 用法：
1. 新建excel表
2. 第一列写不会的东西，例如abandon
3. 第二列写数字，1代表会，2代表不会

|   第一行不写东西         |             |
| :----: | :----: |
| abandon        | 2           |
| ability        | 1           |
| able           | 2           |
|abnormal        | 2           |
|aboard          | 2           |
|about           | 2           |
|above           | 2           |
|abroad          | 2           |
